# medgramVR
Med Gram VR portion 

to get this project to work. 
You must download all required dependencies listed on the json lock file. 
yarn init/ npm init
npm install/ yarn install

npm start/ yarn start

then 

http://localhost:8081/vr/ to get the project working on a browser

Once the browser is you you can port the browser to a mobile phone by entering your computers local ip addres before the localhost address 

for example http://192.168.1.100:8081/vr/index.html.
